Build [Annealer] with given ising model.

Ising model

Ising model is formulated like below: $H = \sum_{i=0}^{len(h)} h_iq_i + \sum_{i,j = 0}^{len(h)} J_{ij}q_iq_j$ $q_i\in{0, 1}$

Example

The settings: $h_i = {1.0 (i = 0), 2.0 (i = 1), 3.0 (i = 2)}$ $J_i = {1.5 (i = 0, j = 1), 2.5 (i = 1, j = 2), 0.0 (otherwise)}$ the code will solve it:

ⓘThis example is not tested
let h = vec![1.0, 2.0, 3.0];
let neighbors = vec![vec![(1, 1.5)], vec![(2, 2.5)], vec![]];

Arguments

  • h - the h factor of Ising model.
  • neighbors - the J factors of Ising model.

Panics

len(h) and len(neighbors) are not the same.
